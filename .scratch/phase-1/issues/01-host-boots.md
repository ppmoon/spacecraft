# 01 — Host boots with tray, launcher, and command palette

**GitHub:** https://github.com/ppmoon/spacecraft/issues/4  
**Parent:** https://github.com/ppmoon/spacecraft/issues/3

**What to build:** A user can launch the Electron TypeScript host on Windows, macOS, or the current test Linux, see a tray/menu-bar presence, open a light launcher, and use the command palette to open a blank OS-level window.

**Blocked by:** None — can start immediately.

**Status:** ready-for-agent

- [ ] App boots as an Electron + TypeScript project with a documented run command
- [ ] Tray / menu-bar entry can show and quit the app
- [ ] Light launcher UI can be opened and closed
- [ ] Command palette (Ctrl/Cmd+K) can open at least one blank OS-level BrowserWindow
- [ ] Smoke check documented or automated for Win, macOS, and current test Linux
